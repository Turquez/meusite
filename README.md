🚀 Sobre o Projeto
Este portfólio foi criado como uma forma centralizada de mostrar meu trabalho e minha jornada profissional. O objetivo é apresentar uma visão clara das minhas competências em desenvolvimento web, design de interfaces e automação de processos.

O design foi pensado para ser moderno, limpo e totalmente responsivo, proporcionando uma ótima experiência em qualquer dispositivo.

💻 Tecnologias Utilizadas
O projeto foi construído utilizando as seguintes tecnologias e bibliotecas:

Estrutura: HTML5 Semântico

Estilização: Tailwind CSS para uma estilização moderna e responsiva.

Interatividade: JavaScript puro (ES6+)

Animações:

Vanta.js: Para o fundo animado de globo interativo.

Swiper.js: Para o carrossel de tecnologias.

Feather Icons: Para os ícones clean e minimalistas.

CSS Animations personalizadas para efeitos como o de digitação (typewriter) e flutuação (floating).

🌟 Funcionalidades
Design Responsivo: Totalmente adaptado para desktops, tablets e smartphones.

Fundo Interativo: Animação de globo 3D que reage aos movimentos do mouse.

Navegação Suave: Scroll suave entre as seções.

Carrossel de Tecnologias: Exibição dinâmica das minhas principais ferramentas de trabalho.

Animações Modernas: Efeitos sutis de hover, animação de digitação e elementos flutuantes para uma experiência mais rica.

Formulário de Contato: Seção de contato para facilitar a comunicação.
